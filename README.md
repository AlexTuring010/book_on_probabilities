# book-on-probabilities

A self-authored study book on probability theory for university classmates, written in Greek with the cheeky title *"What's the probability to pass the exams?"* (Greek: *"Ποια είναι η πιθανότητα να περάσουμε τα μαθήματα;"*). Built as a visual, diagram-heavy refresher to help fellow students understand the material rather than just memorize it.

**[Download or view the PDF →](https://github.com/AlexTuring010/book_on_probabilities/raw/main/probabilities_book.pdf)**

## Contents

The book is in Greek and covers:

- **Introduction to probability theory** — sample spaces, events, axioms, conditional probability
- **Discrete mathematics refresher** — counting, permutations, combinations
- **Discrete and continuous random variables** — distributions, expectation, variance

Heavy on diagrams and worked examples, light on dry formalism — that was the whole point.

## Status — incomplete but useful

Honest disclaimer: this is unfinished. The plan was to add:

- More theoretical depth (the existing prose is example-driven; the formal statements are sometimes hand-wavy)
- A full set of practice problems at the end of each section

I haven't gone back to it in a while and may not. But classmates have used the current version as an exam-prep refresher and found it helpful, so I'm leaving it public as-is.
